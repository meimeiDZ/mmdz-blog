## 看房，选房，砍价

> - 现场看房
>   - **房屋质量方面**，建议到现场查看**有无墙体裂缝、漏水、渗水、偷工减料、管道渗漏**等问题。
>   - 购房者应该上网查发展商资质，查验所选购的住房是否已**被抵押、查封**。
>   - 如果选购的住房是由房地产中介公司代理销售的，应了解该公司是否具有房地产中介资质等。
> - **有无预售资格，查看五证是否齐全。**
